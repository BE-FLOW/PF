# 펫플로우 웹 MVP v0.1

## 제품 목표

보호자가 반려동물의 오늘 상태를 3분 안에 기록하고, 병원 상담에 필요한 사실을 빠짐없이 정리할 수 있게 한다.

## 핵심 사용자 흐름

1. 반려동물 기본 정보를 입력한다.
2. 증상, 식욕, 활력, 지속 기간과 위험 신호를 기록한다.
3. 앱이 관찰 단계와 보호자 행동, 병원 전달용 요약을 만든다.
4. 결과를 이 브라우저에 저장하고 유용성 피드백을 남긴다.

## 이번 버전에 포함

- 모바일 우선 건강 체크 폼
- 한 번 등록하고 재사용하는 반려동물 프로필
- 이름·종류만 필수인 점진적 정보 입력
- 품종 목록 제안, 생일 기반 생애주기 자동 계산, 성별·체중 선택 입력
- `평소와 같아요` 한 번으로 끝내는 빠른 기록
- 규칙 기반 위험 신호 분류
- 선택적 OpenAI 기반 문장 정리
- 병원 전달용 요약과 복사 기능
- 최근 리포트 브라우저 저장
- PWA manifest와 앱 아이콘

## 이번 버전에서 제외

- 회원가입과 클라우드 동기화
- 반려동물 등록번호 수집
- 의료서류 OCR과 이미지 분석
- 동물병원 EMR 연동
- 질병 진단, 약물 및 치료 추천
- 결제, 관리자 대시보드, 푸시 알림
- 방문·체중·검사·예방접종별 복잡한 홈 메뉴

## 안전 원칙

- 위험 단계는 명시적인 로컬 규칙이 결정한다.
- AI는 기록을 정리할 뿐 위험 단계를 낮추거나 진단하지 않는다.
- 호흡 곤란, 의식 저하, 경련, 지속 출혈은 즉시 병원 연락을 안내한다.
- API 키는 서버 Route Handler에서만 사용한다.

## 완료 기준

- 입력에서 리포트까지 모바일 화면에서 완주할 수 있다.
- API 키가 없어도 전체 흐름이 동작한다.
- 새로고침 후 최근 기록을 다시 열 수 있다.
- 저장한 반려동물 정보를 다음 건강 기록에서 다시 묻지 않는다.
- lint, unit test, production build가 통과한다.
