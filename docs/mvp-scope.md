# 펫플로우 테스트 MVP v0.2

## 핵심 흐름

1. 이메일로 가입하고 연락용 휴대전화번호를 포함한 최소 테스터 정보에 동의한다.
2. 여러 반려동물을 등록하고 한 마리를 선택한다.
3. 오늘의 관찰 내용을 입력한다.
4. 안전 규칙 기반 리포트를 확인하고 공유한다.
5. 결과에 짧은 피드백을 남긴다.

## 포함

- 이메일 회원가입과 로그인
- 계정별 다중·다종 반려동물 관리
- 반려동물별 최근 기록 분리
- 건강 체크와 결정론적 위험 분류
- 병원 전달용 요약, 복사, 모바일 공유
- 익명 사용 통계와 테스터 피드백
- 테스터 관리용 Supabase 요약 뷰

## 제외

- 질병 진단, 처방, 약물·용량·치료 추천
- 주소, 정확한 위치, 실명 확인 정보, 반려동물 등록번호
- 결제, 푸시 알림, 동물병원 EMR 연동
- 의료서류 OCR과 이미지 진단
- 일반 사용자용 관리자 화면

## 완료 기준

- 모바일에서 가입부터 리포트까지 완주할 수 있다.
- 로그인 사용자는 본인 반려동물만 조회·수정할 수 있다.
- 선택한 반려동물별로 기록이 분리된다.
- API 키가 없어도 안전 규칙으로 리포트를 생성한다.
- lint, test, build, 원격 배포 검증이 통과한다.
