# 테스트 계정과 반려동물 관리

테스터는 이메일과 비밀번호로 가입하거나 로그인할 수 있다. 가입 시 최소 테스터
정보와 개인정보 수집 동의를 저장한다. 로그인 후에는
여러 종류의 반려동물을 등록하고 한 마리를 선택해 건강 기록을 남긴다.

## 테스트 환경 설정

- 가입 즉시 로그인되도록 이메일 확인을 생략한다.
- 운영 전에는 이메일 확인 또는 Google 로그인을 활성화한다.
- 반려동물 데이터는 `public.pets`에 저장한다.
- RLS 정책으로 로그인 사용자는 본인 반려동물만 조회·수정·삭제할 수 있다.
- 연락처, 주소, 위치 정보는 수집하지 않는다.

## 건강 기록 연결

로그인 상태에서 만든 리포트는 `health_reports.user_id`와
`health_reports.pet_id`에 계정과 선택된 반려동물이 연결된다. 기존 익명 기록은
두 필드가 `null`인 상태로 계속 사용할 수 있다.
